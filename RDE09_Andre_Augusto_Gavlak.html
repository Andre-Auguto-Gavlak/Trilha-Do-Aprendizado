Multimídias

É uma forma de comunicação que combina diferentes formas de conteúdo como texto, áudio ,imagens ,animações ou vídeos em uma única apresentação

Tag IFrame

É usada para incorporar outro documento ao documento HTML.

Incorporar outro site dentro do site

Embutir áudio no site

Tags de Vídeo

É usada para inserir vídeos de um computador local ou de um servidor.

Poster
Define uma imagem estática para o vídeo.
Controls
Define ferramentas de controle para o vídeo.
Source
Faz a busca do endereço e do tipo do vídeo.

Tags de Áudio

É usada para inserir áudios de um computador local ou de um servidor.

Preload
Usado para carregar os metadados(metadata) do arquivo.
Controls
Define ferramentas de controle para o áudio.
Source
Faz a busca do endereço e do tipo do áudio.

Tag Picture

É usada para agrupar imagens. Quando o tamanho de tela é reduzido, a imagem é substituída por uma menor.

Img
Define imagem e descrição.
Source
Define uma imagem e o tamanho máximo de tela em que ela será apresentada.
